# metodo_de_newton_#

## Método de Newton-Raphson Implementação com Python
 
Este projeto implementa o Método de Newton-Raphson para encontrar raízes de funções, permitindo duas abordagens:
1. Método de Newton padrão (com derivada analítica via sympy)
2. Método de Newton aproximado, onde você escolhe explicitamente uma das três aproximações numéricas da derivada:
 * avancada;
 * recuada;
 * central.
Além disso, o código inclui:
* Um modo automático com desafios pré-definidos.
* Um modo interativo onde o usuário pode inserir qualquer função, ponto inicial e tipo de aproximação.
